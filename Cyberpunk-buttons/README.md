# CYBERPUNK INSPIRED BUTTONS

### DESCRIPTION

Cyberpunk inspired buttons that can be used withing web development cycle as a design element. Buttions have an animated glitch effect written in JS.

### TOOLS

In this project I am using Babel transpiler, it's usage in this project in mainly focused on learning about Babel, implementing it in this small scale project.
For study purposes I used offical Babel site:
https://babeljs.io/docs/en/usage

Included package.json in case of any dependencies would be made present or in the future (in case of scaling or upgradig the project).
